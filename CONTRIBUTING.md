# Contributing to A Day in the Life


Thank you for your interest in contributing! This is a narrative project built with Twine and focused on storytelling around marginalized voices.


## How to Contribute


- Fix typos or grammar in passages
- Suggest or implement alternative story branches
- Propose accessibility improvements (text size, contrast, etc.)


## Guidelines


- Be respectful of the story’s tone and purpose
- Avoid over-gamifying — this is a narrative-first experience
- Use inclusive, clear language


## How to Test


1. Clone the repo
2. Open `index.html` in your browser
3. Suggest changes via pull request or submit feedback via Issues


We welcome thoughtful collaboration!
